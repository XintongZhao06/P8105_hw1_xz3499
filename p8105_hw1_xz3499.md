### 4. 绘制散点图

``` r
# 创建散点图
scatter_plot <- ggplot(early_january_weather, aes(x = time_hour, y = temp, color = humid)) +
  geom_point() +
  labs(x = "Time", y = "Temperature (°F)", color = "Humidity") +
  theme_minimal()

# 显示图片
print(scatter_plot)
```
